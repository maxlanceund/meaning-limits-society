本仓库试图回答一个问题：

**当一个社会系统解除了生存强制力之后，它如何持续生成足够多的有效闭环操作？**

旧版《平等制》（`equal-system`）写于意义极限理论成型之前，其中关于“生存压力消除后意义自动转向超越性目标”的假设，在意义极限理论框架下被识别为未经检验的假设，而非推导结论。旧版现作为思想化石保留，不再修改。

本仓库从意义极限理论出发，重新回答同一个问题。

# Meaning Limits Society

This repository attempts to answer one question:

**After a social system removes the survival imperative, how does it continue to generate enough valid closed-loop operations?**

The earlier version, *Equal System* (`equal-system`), was written before the Limits of Meaning theory was fully formed. Its assumption that "meaning will automatically turn toward transcendent goals once survival pressure is removed" is, within the framework of the Limits of Meaning theory, recognized as an untested hypothesis rather than a derived conclusion. The old version is preserved as a fossil of thought and will not be revised.

This repository starts from the Limits of Meaning theory and re-asks the same question from the ground up.
